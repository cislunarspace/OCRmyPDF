# OCRmyPDF

OCRmyPDF adds an OCR text layer to scanned PDFs, making them searchable and selectable.

## Language

**OCR Workflow**:
A user-facing flow that turns an input PDF or image into an output PDF with an OCR text layer. The minimal workflow is selecting input and output files, choosing OCR options, and running OCR.
_Avoid_: Next step, job, task

**Single-file OCR**:
An OCR Workflow for one input file and one output PDF. This is the first GUI workflow before batch directory processing.
_Avoid_: Basic mode, simple mode
